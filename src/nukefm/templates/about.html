About

Prediction-market signals for Bags tokens

nuke.fm tracks Bags token markets and turns their prediction-market pricing into a trader-facing signal. The board starts from Bags token mints, keeps each token linked back to its Bags page, and shows whether the current market signal is live, waiting on seed liquidity, or paused.

Purpose

What the board is for

Bags traders can scan live prediction liquidity, implied terminal price, implied move, token volume, and market cap without leaving the token board. It is a quick read on where the PM thinks a Bags mint may trade by expiry.

Mechanics

How markets work

Each Bags token gets a rolling scalar LONG/SHORT market. The pool price maps to an implied expiry price inside a fixed range around the market’s starting price. Settlement uses stored 24h-median token price snapshots rather than a single spot print.

Access

Read-only web, API trading

The website is read-only. Trading, balances, deposits, withdrawals, and position history live behind the private API. Public pages focus on the Bags board, token context, market state, and chart history.
